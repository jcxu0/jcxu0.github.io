CSS笔记之颜色

发表于 2016-07-07   |  

CSS合法颜色值

可以用下面方法来规定CSS的颜色:
十六进制
RGB
RGBA
HSL
HSLA
预定义/跨浏览器

十六进制

所有浏览器都支持十六进制。
十六进制颜色是这样规定的：#RRGGBB，其中的 RR（红色）、GG（绿色）、BB（蓝色）十六进制整数规定了颜色的成分。所有值必须介于 0 与 FF 之间。

举例说，#0000ff 值显示为蓝色，这是因为蓝色成分被设置为最高值（ff），而其他成分被设置为 0。

RGB

所有的浏览器也都支持RGB颜色值。
RGB 颜色值是这样规定的：rgb(red, green, blue)。每个参数 (red、green 以及 blue) 定义颜色的强度，可以是介于 0 与 255 之间的整数，或者是百分比值（从 0% 到 100%）。
举例说，rgb(0,0,255) 值显示为蓝色，这是因为 blue 参数被设置为最高值（255），而其他被设置为 0。
同样地，下面的值定义了相同的颜色：rgb(0,0,255) 和 rgb(0%,0%,100%)。

RGBA

RGBA 颜色值得到以下浏览器的支持：IE9+、Firefox 3+、Chrome、Safari 以及 Opera 10+。
RGBA 颜色值是 RGB 颜色值的扩展，带有一个 alpha 通道 - 它规定了对象的不透明度。
RGBA 颜色值是这样规定的：rgba(red, green, blue, alpha)。alpha 参数是介于 0.0（完全透明）与 1.0（完全不透明）的数字。

HSL

HSL 颜色值得到以下浏览器的支持：IE9+、Firefox、Chrome、Safari 以及 Opera 10+。
HSL 指的是 hue（色调）、saturation（饱和度）、lightness（亮度） - 表示颜色柱面坐标表示法。
HSL 颜色值是这样规定的：hsl(hue, saturation, lightness)。
Hue 是色盘上的度数（从 0 到 360） - 0 (或 360) 是红色，120 是绿色，240 是蓝色。Saturation 是百分比值；0% 意味着灰色，而 100% 是全彩。Lightness 同样是百分比值；0% 是黑色，100% 是白色。

HSLA

HSLA 颜色值得到以下浏览器的支持：IE9+、Firefox 3+、Chrome、Safari 以及 Opera 10+。
HSLA 颜色值是 HSL 颜色值的扩展，带有一个 alpha 通道 - 它规定了对象的不透明度。
HSLA 颜色值是这样规定的：hsla(hue, saturation, lightness, alpha)，其中的 alpha 参数定义不透明度。alpha 参数是介于 0.0（完全透明）与 1.0（完全不透明）的数字。

以上内容引自 http://www.w3school.com.cn/cssref/css_colors_legal.asp

CSS 笔记之background

发表于 2016-07-07   |  

背景

最近在学一些前端知识，零零散散的，这里记一下。

background大致使用

路径问题

今天在看《CSS Secrets》动手测试的时候出现了url里是本地图片绝对路径地址导致无法显示的问题，具体原因还是不清楚，在网上搜了一下资料，在这里总结一下。

  1. 如果CSS文件是外部链接的话，那么图片的地址应该是相对CSS文件的地址而不是HTML文件的地址。
    例如：

    |-images
|---img.gif
|-css
|---test.css
|-test.html

    那么test.html用test.css文件，在test.css引用img.gif的用法是:background: url(../images/img.gif)

  2. 最好使用相对路径，CSS文件和图片文件不要隔太远。

  3. 给图片路径加引号
    有些老版本的浏览器没法对没有引号的路径进行识别。
